О группе 3ПР

Группа 3ПР обучается по направлению программирования и информационных технологий. Студенты изучают основы веб-разработки, алгоритмы, базы данных и другие дисциплины, необходимые будущему специалисту.

Учебный портал создан для удобного доступа к информации о группе и учебных материалах.

Учебный процесс

В процессе обучения студенты группы 3ПР выполняют лабораторные работы по HTML, CSS и другим веб-технологиям. Каждая лабораторная работа направлена на освоение конкретных практических навыков.

Данный сайт демонстрирует применение всех изученных тем на практике.

Назначение учебного портала

Учебный портал группы 3ПР предназначен для систематизации информации о студентах, лабораторных работах и учебном процессе.

Использование якорной навигации делает работу с длинными страницами более удобной и наглядной.
